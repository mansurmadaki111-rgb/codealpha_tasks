### 📘 [Task 1 — Research Report: IoT in Real Life](./Task1_SmartHomes_Report)

###  [Task 2 — Sensor-Based Simulation](./Task2_PIR_Motion_LED)
A simulated IoT system built on **Tinkercad**, using a **PIR Motion Sensor** to automatically control an LED — turning it on when motion is detected and off when the area is clear. Simulates real-world applications like automatic security lighting and occupancy-based automation.

**Deliverables:** Circuit screenshots, Arduino code, code explanation document

---

###  [Task 3 — IoT Prototype with Hardware](./Task3_SoilMoisture)
A real hardware IoT prototype: a **Smart Soil Moisture Monitoring System** built with an **ESP8266** and a **Soil Moisture Sensor**, transmitting live data over WiFi using the **MQTT protocol** to a custom-built **ThingsBoard Cloud** dashboard. *(Substituted for the Temperature & Humidity Monitoring System task, approved by my mentor, based on available hardware.)*

**Deliverables:** Arduino code, code explanation document, hardware photos, Serial Monitor screenshots, live dashboard screenshot
